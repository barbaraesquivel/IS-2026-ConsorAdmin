# ConsorAdmin: Sistema Integral de Gestión para Consorcios 🏢

**ConsorAdmin** es una plataforma diseñada para digitalizar y optimizar la administración de expensas en edificios de propiedad horizontal. El sistema centraliza la comunicación y gestión entre administradores, consorcistas y proveedores, reemplazando procesos manuales por una solución operativa transparente y eficiente.

## 🚀 ¿Qué es ConsorAdmin?

Es una solución de software orientada a la gestión contable y financiera que permite:

  * **Centralizar la información:** Registro de edificios, unidades funcionales y usuarios.
  * **Control de Expensas:** Seguimiento en tiempo real de deudas, pagos y vencimientos.
  * **Gestión de Gastos:** Registro clasificado de erogaciones y validación de servicios prestados por proveedores.
  * **Transparencia:** Dashboards financieros para la toma de decisiones basadas en datos.

## 🛠️ Prerrequisitos

Para ejecutar y colaborar en este proyecto, necesitas contar con:

  * **Sistema Operativo:** Windows 10 o Windows 11.
  * **Lenguaje de Programación:** C\# (.NET Framework).
  * **Base de Datos:** SQL Server (Gestionado con SQL Server Management Studio 21).
  * **IDE:** Visual Studio 2022 o superior.
